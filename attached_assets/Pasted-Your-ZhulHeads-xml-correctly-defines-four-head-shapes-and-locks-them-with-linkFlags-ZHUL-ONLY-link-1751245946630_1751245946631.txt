Your ZhulHeads.xml correctly defines four head shapes and locks them with <linkFlags>ZHUL_ONLY</linkFlags>, but without a HeadOverlayDef the game never learns about the moustache textures you put in Textures/Things/Pawn/Humanlike/FacialHair.

That’s why nothing turns up when you search for “HeadOverlay” in the mod.

How to add the missing file
Create Defs/HeadOverlayDefs/Zhul_Moustache_HeadOverlays.xml:

xml
Copy
Edit
<?xml version="1.0" encoding="utf-8"?>
<Defs>

  <HeadOverlayDef>
    <defName>Overlay_ZHUL_Moustache</defName>
    <label>curled Zhul moustache</label>

    <!-- one entry for each facing -->
    <southPath>Things/Pawn/Humanlike/FacialHair/Zhul_Moustache_south</southPath>
    <northPath>Things/Pawn/Humanlike/FacialHair/Zhul_Moustache_north</northPath>
    <eastPath>Things/Pawn/Humanlike/FacialHair/Zhul_Moustache_east</eastPath>
    <!-- west is auto-mirrored -->

    <!-- mask to stop bleed-through onto helmets -->
    <maskTex>Things/Pawn/Humanlike/FacialHair/Zhul_Moustache_mask</maskTex>

    <!-- drawSize of 1.0 matches vanilla beard overlays -->
    <drawSize>1.0</drawSize>
  </HeadOverlayDef>

</Defs>
Don’t forget to export Zhul_Moustache_east.png and a matching
Zhul_Moustache_mask.png (64 × 64, black-on-white).

Hook the overlay into your pawns
Add (or merge) this block into every Zhul <PawnKindDef> that should wear the moustache:

xml
Copy
Edit
<headOverlayRecords>
  <li>
    <headOverlay>Overlay_ZHUL_Moustache</headOverlay>
    <chance>1.0</chance>           <!-- 100 % of male Zhuls -->
  </li>
</headOverlayRecords>
If you want only males to get it, patch male PawnKinds or gate it behind a gene / ideology style.

Update About.xml dependency
Because this overlay relies on Humanoid Alien Races and Big & Small scaling, add both to <modDependencies> (right now only Harmony is listed):

xml
Copy
Edit
<modDependencies>
  …
  <li>
    <packageId>erdelf.HumanoidAlienRaces</packageId>
    <displayName>Humanoid Alien Races</displayName>
  </li>
  <li>
    <packageId>RedMattis.BigAndSmallFramework</packageId>
    <displayName>Big &amp; Small Framework</displayName>
  </li>
</modDependencies>
Once the HeadOverlayDefs file is in place and the pawn kinds reference it, the curled moustache will render in game on all four facings—scaled automatically by Big & Small. Let me know if you’d like the full XML for the PawnKind patch or any other code tweaks!
